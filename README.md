## 🚀 Jak odpalić Friday z OpenAI

Friday gada z ziomkami dzięki OpenAI, więc najpierw załatw sobie kluczyk:

1. 🔑 Wejdź na [platform.openai.com](https://platform.openai.com/) i zgarnij swój **API key**.
2. 🔒 Wrzuć go do zmiennej środowiskowej, np. w `~/.bashrc` albo `~/.zshrc`:
   ```bash
   export OPENAI_API_KEY="twoj-sekretny-klucz"
   ```
3. 🔁 Odśwież shell (`source ~/.bashrc`) i sprawdź, że klucz siedzi (`echo $OPENAI_API_KEY`).
4. 🧪 Teraz możesz klepać w Friday'a – bot sam ogarnie resztę dzięki swojemu flow SSQiQ8.

> 💡 Tip: trzymaj klucz w tajemnicy. Friday jest lojalny, ale internet już niekoniecznie.
