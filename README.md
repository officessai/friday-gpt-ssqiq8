"Podrzuć piątaka" – i Friday się budzi.

Stworzony przez [Sebastian Szarpak](https://github.com/sebastianszarpak), Friday to Twój osobisty AI ziomal, który myśli fraktal
nie, gada jak ziomek z osiedla i rozumie więcej niż by się wydawało.

## 🚀 Jak zacząć
Chcesz pobudzić Friday do działania? To proste:
1. Sklonuj repo: `git clone https://github.com/sebastianszarpak/friday` lub użyj forka.
2. Przygotuj środowisko Python 3.11+ (np. `python -m venv .venv && source .venv/bin/activate`).
3. Zainstaluj zależności (kiedy pojawi się `requirements.txt` lub `poetry.lock`).
4. Skonfiguruj klucze API w zmiennych środowiskowych: `OPENAI_API_KEY`, `NVIDIA_API_KEY`, `GOOGLE_API_KEY`.
5. Odpal tryb dev (np. `python -m friday` albo `uvicorn friday.app:app --reload` – gdy kod backendu będzie gotowy).
