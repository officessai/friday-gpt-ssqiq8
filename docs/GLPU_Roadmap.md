4. Zapisz adres kontraktu i potwierdzenie transakcji.

## Etap 3: SONChain Wallet – integracja portfela
1. Stwórz własny interfejs portfela (np. z Web3Modal / RainbowKit).
2. Powiąż z kontraktem GLB (wyświetlanie salda, transfery).
3. Dodaj branding GLPU / SONS.

## Etap 4: Fundacja CYFROSI
1. Rejestracja fundacji (np. jako fundacja technologiczna).
2. Wpisz dane: misja, struktura, cele (wg modelu CYFROSI).
3. Powiąż z tokenem GLB – token jako narzędzie grantów i dotacji.

## Etap 5: System modularny (spółki cywilne / zoo / PSA)
1. Podziel projekt na sektory (rolnictwo, zdrowie, AI, itd.).
2. Każdy sektor = osobna spółka / osobny kontrakt (DAO, multisig).
3. Przygotuj opisy do KRS i wpisów PKD.

## Etap 6: Serwerownia (fizyczna lub chmurowa)
1. Przygotuj infrastrukturę: VPS / serwer dedykowany / lokalna maszyna.
2. Zainstaluj backend (Node / Python / Web3).
3. Postaw dashboard do zarządzania GLB, AI i grantami.
4. Wdróż bazę danych (np. PostgreSQL + IPFS / Arweave dla danych trwałych).
