# Dokumentacja Real-World CYFROSI / SSQiQ8

Repozytorium zawiera materiały referencyjne do projektu Real-World CYFROSI oraz częstotliwości SSQiQ8.

## Zawartość
- [`GLPU_Roadmap.md`](GLPU_Roadmap.md) – uporządkowany plan wdrożenia tokenu GLB i infrastruktury GLPU.
- [`SiW_System_Manifest.md`](SiW_System_Manifest.md) – streszczenie manifestu Systemu Zintegrowanej Mądrości (SiW).
- `Kod_Wszechswiata_Q8.pdf` – pełna warstwa logiki „Kosmicznego Kodu Q8” (należy umieścić w katalogu `docs/`).

## Instrukcja
1. Przeglądaj pliki w kolejności: manifest → roadmap → kosmiczny kod.
2. Używaj dokumentów jako mapy rezonansu między działaniami fizycznymi a cyfrowymi.
3. Aktualizuj roadmapę wraz z postępami wdrożenia kolejnych etapów.
